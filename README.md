🌡️ Temperature Converter

A simple and interactive Temperature Converter web application built using HTML, CSS, and JavaScript. This project allows users to easily convert temperature values between Celsius, Fahrenheit, and Kelvin with a clean and modern UI.

✨ Features

🌡️ Multi-Unit Conversion – Convert between Celsius, Fahrenheit, and Kelvin
⚡ Instant Results – Displays converted values immediately
🎨 Modern UI Design – Gradient background with clean card layout
📱 Responsive Layout – Works on both mobile and desktop
🧮 Accurate Calculations – Uses precise formulas for conversion

🛠️ Technologies Used
HTML5
CSS3
JavaScript
🚀 How to Run the Project

Clone the repository

Open the folder

Run the project:

Double-click index.html
OR
Open with any web browser
🎯 Functionality

⌨️ Enter temperature value → Choose unit (C, F, K)
🔘 Click Convert → Displays results instantly
📊 Shows converted values in two other units
💻 Works smoothly without any page reload
